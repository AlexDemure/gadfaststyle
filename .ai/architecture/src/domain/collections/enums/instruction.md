# Domain Collections: Enums

## Что входит в раздел

- `src/domain/collections/enums/`

## Базовые правила

- Enum размещай по смыслу домена, а не смешивай несвязанные наборы значений в одном файле без причины.
- Имена и значения enum должны повторять существующий стиль проекта.
- Если enum уже используется в схемах, usecase или repository, не меняй его публичную форму без необходимости.
- Новые enum добавляй только тогда, когда в домене действительно есть ограниченный набор допустимых значений.

## Ориентир

Смотри ближайшие enum в `src/domain/collections/enums/` по мере их появления в проекте.
