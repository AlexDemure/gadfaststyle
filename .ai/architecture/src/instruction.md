# Architecture: Src

## Назначение

Эта папка группирует все архитектурные инструкции, которые относятся к коду внутри `src/`.

## Что входит в раздел

- `.ai/architecture/src/http/`
- `.ai/architecture/src/application/`
- `.ai/architecture/src/domain/`
- `.ai/architecture/src/infrastructure/`

## Порядок чтения

1. Прочитай этот файл.
2. Определи, какой слой `src/` затрагивает задача.
3. Перейди в нужный раздел:
   - `.ai/architecture/src/http/instruction.md`
   - `.ai/architecture/src/application/instruction.md`
   - `.ai/architecture/src/domain/instruction.md`
   - `.ai/architecture/src/infrastructure/instruction.md`
4. После этого спускайся по вложенным инструкциям конкретного слоя.
