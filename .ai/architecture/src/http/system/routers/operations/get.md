# HTTP System Router Operation: Get

Используй `get.py` или текущий доменный вариант, если system-ручка читает одну сущность или одно состояние.

- Для чтения одной сущности не используй `search`.
- Handler по имени повторяй по ближайшему system-примеру.
- Endpoint только вызывает usecase и сериализует результат, если это требуется текущим стилем.
