# HTTP System: Dependencies

## Что входит в раздел

- `src/entrypoints/http/system/deps/`

## Правила

- DI-функция лежит рядом с доменом и операцией внутри `system`.
- Называй фабрику `dependency`.
- Никакой сложной логики внутри dependency не нужно, если достаточно вернуть `Usecase()`.
- Если домен уже использует более сложную сборку зависимостей, повторяй текущий паттерн, не вводя новый.

## Ориентир

Смотри ближайшую реализацию в `src/entrypoints/http/system/deps/`.
