# HTTP Public Router Operation: Get

Используй `get.py` или текущий доменный вариант вроде `current.py`, если public-ручка читает одну сущность или одно состояние.

- Для чтения одной сущности не используй `search`.
- Handler по имени повторяй по ближайшему существующему примеру домена.
- Endpoint только вызывает usecase и сериализует результат, если это требуется текущим стилем.
