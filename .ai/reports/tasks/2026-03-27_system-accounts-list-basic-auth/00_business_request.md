# Business Request

## Исходная постановка

Сделай ручку список пользователей с доступом по http-basic для администраторов в system папку.

## Интерпретация в рамках текущего проекта

- Под пользователями в текущем проекте понимаются `Account`, потому что отдельной сущности `User` в домене нет.
- Ручка должна жить в `src/entrypoints/http/system/`.
- Доступ должен быть ограничен через существующий dependency `basic`.
