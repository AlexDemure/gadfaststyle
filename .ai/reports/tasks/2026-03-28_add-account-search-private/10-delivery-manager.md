# Delivery Manager

## Request

`Задача: Добавь ручку :search в приватные ручки для сущности accounts с пагинацией с сортировкой по полю created, с возможность desc\asc и точному фильтру по полю id.`

## Scope

- Добавить приватную HTTP-ручку `accounts:search`.
- Провести изменение по конвейеру отчетов.
- Подтвердить кодом пагинацию, сортировку по `created` и точный фильтр по `id`.

## Handoff

- `20-project-architect` должен определить точки расширения в `system` HTTP-контуре и тестовом зеркале.
- `30-task-orchestrator` должен собрать пошаговый технический план.
- `40-code-implementer` должен добавить production-код.
- `50-test-writer` должен добавить интеграционные тесты.
- `60-code-reviewer` должен проверить риски и пробелы.
- `70-report-compiler` должен собрать финальный отчет с полной историей.
