## Назначение

`report-compiler` собирает финальный отчет задачи из артефактов pipeline.

## Файлы пакета

- [RULES.MD](./RULES.MD) - правила для `report-compiler`.
- [INPUT.MD](./INPUT.MD) - ожидаемый вход этапа.
- [OUTPUT.MD](./OUTPUT.MD) - обязательный результат этапа.
- [SKILL.MD](./SKILL.MD) - рабочий playbook роли.
