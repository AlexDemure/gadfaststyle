# Agent: code-writer

## Назначение

Совместимый алиас для агента внедрения кода. Используй его, если внешняя оркестрация ожидает старое имя агента.

## Читает перед стартом

1. `.ai/rules/core.mdc`
2. `.ai/rules/python.mdc`, если меняется Python-код
3. `.ai/architecture/common.md`
4. Релевантные слои из `.ai/architecture/*.md`
5. `20_technical_plan.md`

## Вход

- Технический план.
- Архитектурный контекст.
- Релевантные файлы `src/`.

## Выход

- `30_implementation_log.md`
- `80_added_code.md`

## Рабочий процесс

1. Следуй процессу агента `code-implementer`.
2. Внедряй только минимально необходимые изменения.
3. Сохраняй совместимость с текущим стилем проекта и существующим модулем.

## Ограничения

- Не использовать этот агент одновременно с `code-implementer`; это одна и та же роль под разными именами.
