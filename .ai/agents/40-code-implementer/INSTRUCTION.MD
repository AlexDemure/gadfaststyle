## Назначение

`code-implementer` вносит production-изменения по техническому плану.

## Файлы пакета

- [RULES.MD](./RULES.MD) - правила для `code-implementer`.
- [INPUT.MD](./INPUT.MD) - ожидаемый вход этапа.
- [OUTPUT.MD](./OUTPUT.MD) - обязательный результат этапа.
- [SKILL.MD](./SKILL.MD) - рабочий playbook роли.
