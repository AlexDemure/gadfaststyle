## Назначение

`spec` описывает machine-readable контракт агентного runtime.

## Файлы пакета

- [pipeline.yaml](./pipeline.yaml) - graph pipeline и переходы.
- [agents.yaml](./agents.yaml) - registry агентов и их runtime-метаданные.
- [artifacts.yaml](./artifacts.yaml) - контракты артефактов.
- [knowledge.yaml](./knowledge.yaml) - декларативный bundle knowledge-документов для engine.
- [models.yaml](./models.yaml) - модели и provider-конфигурация.
- [settings.yaml](./settings.yaml) - лимиты и runtime-политики.
- [schemas/pipeline.schema.json](./schemas/pipeline.schema.json) - JSON schema pipeline spec.
- [schemas/agents.schema.json](./schemas/agents.schema.json) - JSON schema agents spec.
- [schemas/artifacts.schema.json](./schemas/artifacts.schema.json) - JSON schema artifacts spec.
- [schemas/knowledge.schema.json](./schemas/knowledge.schema.json) - JSON schema knowledge spec.
- [schemas/models.schema.json](./schemas/models.schema.json) - JSON schema models spec.
