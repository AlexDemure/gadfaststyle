## Назначение

`tests` описывает тестовый слой.

## Файлы пакета

- [rules.md](./rules.md) - правила для тестов.
- [test_integrations/instruction.md](./test_integrations/instruction.md) - инструкции для интеграционных тестов.
- [test_units/instruction.md](./test_units/instruction.md) - инструкции для unit-тестов.
- [factories/instruction.md](./factories/instruction.md) - инструкции для тестовых фабрик.
- [mocking/instruction.md](./mocking/instruction.md) - инструкции для тестовых mock-объектов.
