## Назначение

`utils` описывает переиспользуемые прикладные утилиты.

## Файлы пакета

- [dependencies.md](./dependencies.md) - обязательные зависимости для `application/utils`.
- [rules.md](./rules.md) - правила для прикладных утилит.
- [examples.md](./examples.md) - кодовые эталоны формы `application/utils`.
