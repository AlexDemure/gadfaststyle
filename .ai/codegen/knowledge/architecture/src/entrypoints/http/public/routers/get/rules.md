## Правила

- Используй `get.py` или доменный вариант вроде `current.py`, если ручка читает одну сущность или одно состояние.
- Для чтения одной сущности не используй `search`.
- Handler по имени повторяй по принятому паттерну домена.
- Endpoint только вызывает usecase и сериализует результат, если это требуется контрактом.
