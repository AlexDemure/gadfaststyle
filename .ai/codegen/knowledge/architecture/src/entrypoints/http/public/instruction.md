## Назначение

`public` описывает публичный HTTP-контур.

## Файлы пакета

- [rules.md](./rules.md) - общие правила для `public`.
- [routers/instruction.md](./routers/instruction.md) - инструкции для public routers.
- [deps/instruction.md](./deps/instruction.md) - инструкции для public deps.
- [schemas/instruction.md](./schemas/instruction.md) - инструкции для public schemas.
