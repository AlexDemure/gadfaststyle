## Правила

- Enum размещай по смыслу домена.
- Имена и значения enum должны следовать принятому стилю домена.
- Не меняй публичную форму enum без необходимости, если он уже используется в других слоях.
- Добавляй enum только для реально ограниченного набора значений.
- Публичные enum экспортируй через `enums/__init__.py` с явными импортами и `__all__`.
