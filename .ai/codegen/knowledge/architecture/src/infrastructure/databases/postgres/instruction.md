## Назначение

`postgres` описывает Postgres-реализацию database-инфраструктуры.

## Файлы пакета

- [rules.md](./rules.md) - общие правила для Postgres-инфраструктуры.
- [tables/instruction.md](./tables/instruction.md) - инструкции для SQLAlchemy tables.
- [crud/instruction.md](./crud/instruction.md) - инструкции для CRUD-слоя.
- [adapters/instruction.md](./adapters/instruction.md) - инструкции для repository adapters.
