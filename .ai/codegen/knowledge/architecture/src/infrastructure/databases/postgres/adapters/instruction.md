## Назначение

`adapters` описывает repository adapters для Postgres.

## Файлы пакета

- [dependencies.md](./dependencies.md) - обязательные зависимости для repository adapters.
- [rules.md](./rules.md) - правила для repository adapters.
- [examples.md](./examples.md) - кодовые эталоны формы repository adapter.
