## Назначение

`infrastructure` описывает инфраструктурный слой.

- [rules.md](./rules.md) - общие правила для infrastructure-слоя.
- [databases/instruction.md](./databases/instruction.md) - инструкции для database-инфраструктуры.
- [security/instruction.md](./security/instruction.md) - инструкции для security-клиентов.
- [integrations/instruction.md](./integrations/instruction.md) - инструкции для внешних интеграций.
- [storages/instruction.md](./storages/instruction.md) - инструкции для storage-клиентов.
- [monitoring/instruction.md](./monitoring/instruction.md) - инструкции для monitoring-пакетов.
