## Назначение

Используй `get`, когда сценарий читает одну сущность или одно состояние по конкретному идентификатору, ключу или типу доступа.

## Файлы пакета

- [../../RULES.MD](../../RULES.MD) - общие правила для всех usecase.
- [../INSTRUCTION.MD](../INSTRUCTION.MD) - общие правила для query-usecase.
- [DEPENDENCIES.MD](./DEPENDENCIES.MD) - обязательные зависимости `get`-usecase.
- [RULES.MD](./RULES.MD) - правила для `get`.
- [EXAMPLES.MD](./EXAMPLES.MD) - кодовые эталоны формы `get`.
