## Назначение

Используй `delete`, когда сценарий удаляет сущность или переводит ее в удаленное состояние.

- [../../RULES.MD](../../RULES.MD) - общие правила для всех usecase.
- [../INSTRUCTION.MD](../INSTRUCTION.MD) - общие правила для command-usecase.
- [DEPENDENCIES.MD](./DEPENDENCIES.MD) - обязательные зависимости `delete`-usecase.
- [RULES.MD](./RULES.MD) - правила для `delete`.
- [EXAMPLES.MD](./EXAMPLES.MD) - кодовые эталоны формы `delete`.
