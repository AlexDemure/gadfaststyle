## Назначение

Используй `create`, когда сценарий создает новую сущность или новую связь.

## Файлы пакета

- [../../RULES.MD](../../RULES.MD) - общие правила для всех usecase.
- [../INSTRUCTION.MD](../INSTRUCTION.MD) - общие правила для command-usecase.
- [DEPENDENCIES.MD](./DEPENDENCIES.MD) - обязательные зависимости `create`-usecase.
- [RULES.MD](./RULES.MD) - правила для `create`-сценариев.
- [EXAMPLES.MD](./EXAMPLES.MD) - кодовые эталоны формы `create`-usecase.
