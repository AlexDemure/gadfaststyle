## Назначение

Используй `update`, когда сценарий изменяет существующую сущность или часть ее состояния.

## Файлы пакета

- [../../RULES.MD](../../RULES.MD) - общие правила для всех usecase.
- [../INSTRUCTION.MD](../INSTRUCTION.MD) - общие правила для command-usecase.
- [DEPENDENCIES.MD](./DEPENDENCIES.MD) - обязательные зависимости `update`-usecase.
- [RULES.MD](./RULES.MD) - правила для `update`.
- [EXAMPLES.MD](./EXAMPLES.MD) - кодовые эталоны формы `update`.
