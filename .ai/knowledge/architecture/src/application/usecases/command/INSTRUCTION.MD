## Назначение

`command` описывает mutation-сценарии: `create`, `update`, `delete` и другие сценарии записи.

## Файлы пакета

- [RULES.MD](./RULES.MD) - общие правила для command-usecase.
- [create/INSTRUCTION.MD](./create/INSTRUCTION.MD) - правила для `create`.
- [delete/INSTRUCTION.MD](./delete/INSTRUCTION.MD) - правила для `delete`.
- [update/INSTRUCTION.MD](./update/INSTRUCTION.MD) - правила для `update`.
