## Назначение

`collections` описывает доменные коллекции.

## Файлы пакета

- [DEPENDENCIES.MD](./DEPENDENCIES.MD) - обязательные зависимости для domain collections.
- [RULES.MD](./RULES.MD) - правила для domain collections.
- [exceptions/INSTRUCTION.MD](./exceptions/INSTRUCTION.MD) - инструкции для доменных исключений.
- [enums/INSTRUCTION.MD](./enums/INSTRUCTION.MD) - инструкции для доменных enum.
