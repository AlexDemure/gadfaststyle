## Назначение

`http` описывает HTTP entrypoint.

## Файлы пакета

- [rules.md](./rules.md) - общие правила для HTTP-слоя.
- [common/instruction.md](./common/instruction.md) - инструкции для общего HTTP-кода.
- [public/instruction.md](./public/instruction.md) - инструкции для публичного контура.
- [system/instruction.md](./system/instruction.md) - инструкции для внутреннего контура.
