## Назначение

`system` описывает внутренний HTTP-контур.

## Файлы пакета

- [RULES.MD](./RULES.MD) - общие правила для `system`.
- [routers/INSTRUCTION.MD](./routers/INSTRUCTION.MD) - инструкции для system routers.
- [deps/INSTRUCTION.MD](./deps/INSTRUCTION.MD) - инструкции для system deps.
- [schemas/INSTRUCTION.MD](./schemas/INSTRUCTION.MD) - инструкции для system schemas.
