## Правила

- Используй `get.py` или текущий доменный вариант, если system-ручка читает одну сущность или одно состояние.
- Для чтения одной сущности не используй `search`.
- Handler по имени повторяй по принятому паттерну `system`.
- Endpoint только вызывает usecase и сериализует результат, если это требуется контрактом.
