## Правила

- Схемы одного домена группируй в одном модуле, если пакет уже следует этой форме.
- Для request используй базовые классы из `http/common`.
- Для response используй `Response`.
- Имена схем строй как `<Operation><Entity>`.
- Для search-запросов используй явную структуру `filters`, `sorting`, `pagination`.
- Публичные схемы экспортируй через `schemas/__init__.py` с явными импортами и `__all__`, чтобы использовать импорт вида `from ...schemas import CurrentAccount`.
