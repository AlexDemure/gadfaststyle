## Правила

- В каждом файле операции создавай `router = APIRouter()`.
- Не объединяй несколько операций в один router-файл.
- Обработчик называй `command` для mutation и `query` для read-ручек, если пакет не задает иной паттерн.
- Endpoint принимает схему, вызывает usecase и не знает о БД и инфраструктуре напрямую.
- Формат URL должен следовать текущему паттерну контура.
- Новый домен подключай через `registry.py`.
