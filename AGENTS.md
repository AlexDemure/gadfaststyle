## Агентные инструкции проекта

Этот репозиторий содержит две сущности:

- **Инструкции по написанию кода (правила)**: `.ai/knowledge/rules/*.MDC`
- **Шаблоны промптов для будущих агентов**: `.ai/agents/*`

### Как агенту работать с репозиторием

- Если пользователь начинает сообщение с `Задача:`, сначала прочитай `.ai/knowledge/rules/CORE.MDC`.
- Если пользователь начинает сообщение с `Задача:` и работа идет с Python, дополнительно прочитай `.ai/knowledge/rules/PYTHON.MDC`.
- Если пользователь начинает сообщение с `Задача:`, используй материалы из `.ai/knowledge/` и `.ai/agents/`, запусти полный агентный конвейер и веди артефакты в `.ai/runtime/tasks/`.
- Если префикса `Задача:` нет, считай сообщение прямой командой без обязательного запуска конвейера и без обязательного чтения `.ai/knowledge/` и `.ai/agents/`.
- В любом режиме сначала найди и прочитай релевантные файлы в репозитории, потом вноси минимальные правки.

### Где лежат инструкции

- **Правила**: `.ai/knowledge/rules/`
  - `CORE.MDC`: общие требования для режима `Задача:`
  - `PYTHON.MDC`: Python-стандарты для режима `Задача:` при работе с `**/*.py`
- **Будущие агенты**: `.ai/agents/`
