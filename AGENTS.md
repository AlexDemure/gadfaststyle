## AGENTS

- `src/` — исходный код сервиса.
- `tests/` — тесты проекта.
- `.instructions/` — общие инструкции проекта: архитектура, структура папок, code style и правила по слоям.
- `.specs/` — path-based `spec.md`, которые зеркалят проект и описывают, как должны работать файлы в `src/` и `tests/`.
- `.codex/` — рабочая папка OpenAI/Codex с ролями агентов, командами и runtime-артефактами.
