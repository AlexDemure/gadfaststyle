# Инструкции проекта

Перед изменениями читай только релевантный документ:

| Файл              | Когда читать            |
|-------------------|-------------------------|
| `ARCHITECTURE.md` | Любая работа в `src/`   |
| `CODESTYLE.md`    | Любой Python-код        |
| `TESTING.md`      | Любая работа в `tests/` |

`README.md` — пользовательский документ. Для реализации кода и тестов ориентируйся на специализированные агентные `.md` файлы.

# Зоны ответственности

| Папка       | Назначение                       | Правила                            |
|-------------|----------------------------------|------------------------------------|
| `src/`      | production-код                   | `ARCHITECTURE.md` + `CODESTYLE.md` |
| `tests/`    | endpoint-тесты, factories, mocks | `TESTING.md`                       |
| `.scripts/` | служебные скрипты                | не менять без явной задачи         |
| `.compose/` | инфраструктура                   | не менять без явной задачи         |

# Игнорировать

Не читать и не менять:

`.venv/` `.volumes/` `__pycache__/` `.git/` `.mypy_cache/` `.ruff_cache/` `.pytest_cache/`
