## Агентные инструкции проекта

Этот репозиторий содержит две сущности:

- **Инструкции по написанию кода (правила)**: `.ai/knowledge/rules/*.mdc`
- **Шаблоны промптов для будущих агентов**: `.ai/knowledge/agents/*`

### Как агенту работать с репозиторием

- **Всегда** прочитай `.ai/knowledge/rules/core.mdc`.
- Если работаешь с Python, прочитай `.ai/knowledge/rules/python.mdc`.
- Затем действуй: сначала найди и прочитай релевантные файлы в репозитории, потом вноси минимальные правки.
- Если пользователь начинает сообщение с `Задача:`, запусти полный агентный конвейер и веди артефакты в `.ai/runtime/tasks/`.
- Если префикса `Задача:` нет, считай сообщение прямой командой без обязательного запуска конвейера.

### Где лежат инструкции

- **Правила**: `.ai/knowledge/rules/`
  - `core.mdc`: общие требования (применяются всегда)
  - `python.mdc`: Python-стандарты (применяются для `**/*.py`)
- **Будущие агенты**: `.ai/knowledge/agents/`
