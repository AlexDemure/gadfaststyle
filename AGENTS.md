## Агентные инструкции проекта

Этот репозиторий содержит несколько сущностей:

- **Инструкции по написанию кода (правила)**: `.instructions/rules/*.MDC`
- **Общая база знаний проекта**: `.instructions/*`
- **CodeSpeak-промты для генерации кода**: `.specs/*`
- **Инструкции для Codex-агентов**: `.codex/agents/*`

### Как агенту работать с репозиторием

- Если пользователь начинает сообщение с `Задача:`, сначала прочитай `.instructions/rules/README.md`.
- Если пользователь начинает сообщение с `Задача:` и работа идет с Python, дополнительно прочитай Python-раздел в `.instructions/rules/README.md`.
- Если пользователь начинает сообщение с `Задача:`, используй материалы из `.instructions/`, `.specs/` и `.codex/agents/`, запусти полный агентный конвейер и веди артефакты в `.codex/tasks/`.
- Если префикса `Задача:` нет, считай сообщение прямой командой без обязательного запуска конвейера и без обязательного чтения `.instructions/`, `.specs/` и `.codex/agents/`.
- В любом режиме сначала найди и прочитай релевантные файлы в репозитории, потом вноси минимальные правки.

### Где лежат инструкции

- **Правила**: `.instructions/rules/`
  - `README.md`: общие требования для режима `Задача:` и Python-стандарты для `**/*.py`
- **CodeSpeak**: `.specs/`
- **Codex-агенты**: `.codex/agents/`
