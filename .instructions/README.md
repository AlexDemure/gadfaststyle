## Описание

`.instructions` хранит правила структуры проекта, слоев, импортов и стиля кода.

## Правила

- Общие правила production-кода лежат в `src/README.md`.
- Общие правила тестов лежат в `tests/README.md`.
- Формат `spec.md` лежит в `specs/README.md`.
- Внутри локальных README описывай только правила своей области, без пайплайна и ролей.

## Примеры

```text
.instructions/
├── src/
├── tests/
└── specs/
```
