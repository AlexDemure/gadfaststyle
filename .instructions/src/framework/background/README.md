## Описание

`framework/background` описывает фоновые клиенты рантайма и их настройку.

## Правила

- В `background` держи только фоновые клиенты и их жизненный цикл.
- Клиенты с запуском и остановкой оформляй явно.
- Не размещай здесь бизнесовые обработчики задач.

## Примеры

```python
from src.framework.background import client
```
