## Описание

`framework/openapi/models` описывает модели OpenAPI-слоя.

## Правила

- Здесь держи только модели конфигурации и представления OpenAPI.
- Модели должны быть техническими и независимыми от domain.
- Публичные модели экспортируй через пакет.

## Примеры

```python
from src.framework.openapi.models import affix
```
