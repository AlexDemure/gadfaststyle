## Описание

`framework/routing/collections` описывает коллекции routing-слоя.

## Правила

- В `collections` держи только общие наборы значений routing-каркаса.
- Enum раскладывай по подпакетам.
- Не размещай здесь router-реализацию.

## Примеры

```python
from src.framework.routing.collections import enums
```
