## Описание

`static` описывает статические ресурсы и пакеты без логики рантайма.

## Правила

- В `static` не размещай Python-логику приложения.
- Используй этот пакет только для ресурсов, константных файлов или пустых package-маркеров, если они нужны структуре проекта.
- Не добавляй сюда `__init__.py`, если пакет не должен быть импортируемым по правилам шаблона.

## Примеры

```text
src/static/
```
