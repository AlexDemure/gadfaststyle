## Описание

`os` описывает общие OS-утилиты и системные константы.

## Правила

- В `os` держи только технический код без бизнесовой логики.
- Константы раскладывай в `collections/`, функции в `utils/`.
- Не привязывай утилиты к одному конкретному usecase.

## Примеры

```python
from src.common.os.utils import memory
```
