## Описание

`os/collections` описывает наборы констант системного слоя.

## Правила

- В `collections` держи только системные константы и фиксированные наборы значений.
- Не размещай здесь функции и доступ к ОС.
- Публичные элементы экспортируй через пакет.

## Примеры

```python
from src.common.os.collections import const
```
