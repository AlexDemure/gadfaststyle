## Описание

`os/utils` описывает системные утилиты.

## Правила

- Каждая утилита решает одну техническую задачу работы с ОС.
- Не размещай здесь бизнесовую логику и инфраструктурные клиенты.
- Побочные эффекты делай явными в названии и сигнатуре.

## Примеры

```python
from src.common.os.utils import memory
```
