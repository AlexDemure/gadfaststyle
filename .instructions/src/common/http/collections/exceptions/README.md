## Описание

`http/collections/exceptions` описывает общие HTTP-исключения.

## Правила

- Ошибки должны отражать транспортный слой, а не бизнесовую модель.
- Каждое исключение описывает один сценарий сбоя.
- Не смешивай здесь domain- и infrastructure-ошибки.

## Примеры

```python
class HTTPError(Exception):
    pass
```
