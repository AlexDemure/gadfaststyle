## Описание

`formats/encoders` описывает кодеки и сериализацию общих форматов.

## Правила

- Каждый encoder отвечает за один формат или один способ сериализации.
- Encoder не должен тянуть бизнесовые типы и предметные зависимости.
- Общие encoder-функции делай переиспользуемыми из других слоев.

## Примеры

```python
from src.common.formats.encoders import json
```
