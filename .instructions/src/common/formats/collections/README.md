## Описание

`formats/collections` описывает наборы констант и enum для общих форматов.

## Правила

- В `collections` держи только фиксированные наборы значений и ошибки слоя `formats`.
- Не размещай здесь функции, кодеки и утилит рантаймаы.
- Публичные элементы экспортируй через пакет.

## Примеры

```python
from src.common.formats.collections import enums
```
