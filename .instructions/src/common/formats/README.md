## Описание

`formats` описывает общие преобразования данных и утилиты форматов.

## Правила

- В `formats` держи только переиспользуемые преобразования без бизнесовой логики.
- Константы и enum раскладывай в `collections/`, кодеки в `encoders/`, утилиты в `utils/`.
- Публичные импорты собирай через пакет, если это уже принято в модуле.

## Примеры

```python
from src.common.formats.utils import json
```
