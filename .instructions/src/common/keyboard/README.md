## Описание

`keyboard` описывает общие клавиатурные и символьные наборы.

## Правила

- В `keyboard` держи только общие наборы символов и раскладок.
- Константы раскладывай в `collections/`.
- Не добавляй сюда ввод-вывод и OS-зависимую логику.

## Примеры

```python
from src.common.keyboard import collections
```
