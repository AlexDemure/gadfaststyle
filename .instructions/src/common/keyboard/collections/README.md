## Описание

`keyboard/collections` описывает фиксированные наборы символов и раскладок.

## Правила

- В `collections` держи только константные наборы.
- Для вычисляемых преобразований используй отдельные утилиты вне этого пакета.
- Публичные элементы экспортируй через пакет.

## Примеры

```python
from src.common.keyboard.collections import const
```
