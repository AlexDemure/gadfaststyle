## Описание

`typings/variables` описывает общие переменные и шаблоны ошибок слоя типов.

## Правила

- В `variables` держи только переиспользуемые значения без логики.
- Не смешивай переменные с валидаторами и функциями.
- Имена должны быть понятны без чтения конкретного usecase.

## Примеры

```python
from src.common.typings.variables import error
```
