## Описание

`typings` описывает общие типы, валидаторы и переменные ошибок.

## Правила

- В `typings` держи только общие типы и проверки, используемые в разных слоях.
- Валидаторы раскладывай в `validators/`, общие переменные и сообщения в `variables/`.
- Не добавляй сюда предметные модели и domain-ошибки.

## Примеры

```python
from src.common.typings.validators import string
```
