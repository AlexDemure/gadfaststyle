## Описание

`decorators/usecases` описывает декораторы прикладного слоя.

## Правила

- Здесь держи только декораторы, которые оборачивают usecase-вызовы.
- Декоратор не должен менять бизнесовые правила usecase.
- Обертки с сессией, ретраями и похожими cross-cutting задачами держи здесь.

## Примеры

```python
from src.decorators.usecases.session import session
```
