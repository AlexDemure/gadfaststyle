## Описание

`storages/redis` описывает Redis-инфраструктуру проекта.

## Правила

- В `redis` держи только техническую работу с Redis-клиентом и его контрактами.
- Коллекции enum и исключений раскладывай по подпакетам.
- Redis-слой не должен содержать бизнесовую логику.

## Примеры

```python
from src.infrastructure.storages import redis
```
