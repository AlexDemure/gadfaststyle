## Описание

`logging/models` описывает модели logging-слоя.

## Правила

- Здесь держи только технические модели логов и их представления.
- Модели должны быть независимы от предметной сущности.
- Публичные модели экспортируй через пакет.

## Примеры

```python
from src.infrastructure.monitoring.logging import models
```
