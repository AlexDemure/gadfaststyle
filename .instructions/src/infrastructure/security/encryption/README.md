## Описание

`security/encryption` описывает шифрование и дешифрование данных.

## Правила

- В `encryption` держи только техническую реализацию шифрования.
- Коллекции ошибок и наборов значений раскладывай по подпакетам.
- Шифрование не должно знать о бизнесовой сущности.

## Примеры

```python
from src.infrastructure.security import encryption
```
