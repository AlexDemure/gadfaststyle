## Описание

`jwt/models` описывает модели JWT-слоя.

## Правила

- Здесь держи только технические модели payload и token-данных.
- Модели должны быть независимы от предметной сущности.
- Публичные модели экспортируй через пакет.

## Примеры

```python
from src.infrastructure.security.jwt import models
```
