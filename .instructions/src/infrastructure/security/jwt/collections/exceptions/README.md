## Описание

`jwt/collections/exceptions` описывает исключения JWT-слоя.

## Правила

- Исключения должны описывать только ошибки токенов и их проверки.
- Не смешивай их с transport- и domain-ошибками.
- Каждое исключение описывает один тип JWT-сбоя.

## Примеры

```python
class JWTError(Exception):
    pass
```
