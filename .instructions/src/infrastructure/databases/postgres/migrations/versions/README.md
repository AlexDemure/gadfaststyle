## Описание

`postgres/migrations/versions` описывает версии миграций.

## Правила

- Каждая миграция должна быть изолированным изменением схемы.
- Не смешивай в одном файле несколько несвязанных изменений.
- Имена и ревизии должны быть согласованы с Alembic.

## Примеры

```text
src/infrastructure/databases/postgres/migrations/versions/
```
