## Описание

`sqlalchemy/profilers` описывает профилирование ORM-запросов.

## Правила

- Профилировщики должны быть техническими и переиспользуемыми.
- Не добавляй сюда бизнесовую интерпретацию метрик.
- Сбор профиля не должен ломать основной поток выполнения.

## Примеры

```python
from src.infrastructure.databases.orm.sqlalchemy import profilers
```
