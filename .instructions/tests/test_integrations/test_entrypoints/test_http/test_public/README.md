## Описание

`test_http/test_public` описывает интеграционные тесты публичных HTTP-ручек.

## Правила

- Здесь проверяй публичные маршруты и пользовательские сценарии.
- Тесты должны идти через public-контур и его auth/deps.
- Не смешивай здесь system-маршруты.

## Примеры

```python
request = await self.client.get("/api/accounts:get")
```
