## Описание

`test_integrations/test_entrypoints/test_http` описывает HTTP-интеграционные тесты.

## Правила

- HTTP-тесты должны ходить через реальный HTTP-контур приложения.
- Используй принятые фикстуры клиента и сессии.
- Проверяй и код ответа, и эффект в системе.
