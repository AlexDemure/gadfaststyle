## Описание

`test_integrations/test_entrypoints` описывает интеграционные тесты entrypoint-слоя.

## Правила

- Здесь проверяй транспортный контур и его интеграцию с приложением.
- Тесты размещай зеркально относительно entrypoint-структуры.
- Не превращай эти тесты в unit-тесты внутренних функций.

```python
from httpx import AsyncClient
```
