## Описание

`factories/domain/models` описывает фабрики domain-моделей.

## Правила

- Каждая фабрика должна быстро создавать валидную domain-модель.
- Значения по умолчанию делай пригодными для большинства тестов.
- Не добавляй в фабрику инфраструктурный доступ и побочные эффекты.

## Примеры

```python
from tests.factories.domain.models import Account
```
