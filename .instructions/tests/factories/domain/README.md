## Описание

`factories/domain` описывает фабрики domain-объектов для тестов.

## Правила

- Фабрики этого слоя создают domain-модели и значения без инфраструктурных побочных эффектов.
- Каждая domain-группа фабрик раскладывается по подпакетам.
- Не смешивай здесь HTTP- и database-фикстуры.

## Примеры

```python
from tests.factories.domain.models import Account
```
