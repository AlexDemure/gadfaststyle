## Описание

`tools/profiler/profilers/extensions` описывает расширения профилировщиков.

## Правила

- Каждое расширение добавляет одну техническую возможность профилировщику.
- Не смешивай расширения с моделями и отчетами.
- Держи API расширений узким и предсказуемым.

## Примеры

```python
from tests.tools.profiler.profilers import extensions
```
