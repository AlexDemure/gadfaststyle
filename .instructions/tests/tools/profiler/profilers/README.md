## Описание

`tools/profiler/profilers` описывает реализации профилировщиков.

## Правила

- Каждый профилировщик отвечает за один способ измерения или один контур выполнения.
- Расширения профилировщиков раскладывай по подпакетам.
- Профилировщики не должны содержать бизнесовую логику.

## Примеры

```python
from tests.tools.profiler import profilers
```
