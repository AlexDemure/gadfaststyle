## Описание

`tools/profiler/reports` описывает генерацию отчетов профилировщика.

## Правила

- Здесь держи только код генерации и сборки отчетов.
- Форматы отчетов и расширения раскладывай по подпапкам.
- Генерация отчетов не должна содержать бизнесовую интерпретацию.

## Примеры

```python
from tests.tools.profiler import reports
```
