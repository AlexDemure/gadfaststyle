## Описание

`tools/profiler/reports/python` описывает Python-отчеты профилировщика.

## Правила

- Здесь держи генерацию Python-specific отчетов.
- Не смешивай Python-формат с общими утилитами отчетов.
- Отчеты должны быть детерминированными и читаемыми.

## Примеры

```python
from tests.tools.profiler.reports import python
```
