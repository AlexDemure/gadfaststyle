## Описание

`tools/profiler/reports/extensions` описывает расширения генерации отчетов.

## Правила

- Каждое расширение добавляет одну техническую возможность слою отчетов.
- Не смешивай расширения с моделями отчетов.
- Держи API расширений узким и предсказуемым.

## Примеры

```python
from tests.tools.profiler.reports import extensions
```
