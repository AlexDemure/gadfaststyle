## Описание

`tools/profiler/contextmanagers` описывает контекстные менеджеры профилировщика.

## Правила

- Каждый контекстный менеджер должен инкапсулировать один профильный сценарий.
- Не смешивай здесь модели отчетов и парсинг результатов.
- Контекстный менеджер должен быть пригоден для повторного использования.

## Примеры

```python
from tests.tools.profiler import contextmanagers
```
