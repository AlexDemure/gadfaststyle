## Описание

`tools/profiler/models` описывает модели профилировщика.

## Правила

- Здесь держи только технические модели профиля и отчета.
- Модели должны быть независимы от конкретной фичи.
- Публичные модели экспортируй через пакет.

## Примеры

```python
from tests.tools.profiler.models import reports
```
