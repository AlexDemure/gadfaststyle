## Описание

`tools/profiler/models/profilers` описывает модели профилировщиков.

## Правила

- Здесь держи структуры данных самих профилировщиков.
- Расширения моделей раскладывай в `extensions/`.
- Не смешивай модели с реализацией рантайма профилировщика.

## Примеры

```python
from tests.tools.profiler.models import profilers
```
