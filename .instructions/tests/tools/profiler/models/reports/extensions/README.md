## Описание

`tools/profiler/models/reports/extensions` описывает расширения моделей отчетов.

## Правила

- Расширения должны быть техническими и узкими по задаче.
- Не добавляй сюда генерацию во время выполнения отчетов.
- Каждое расширение решает одну задачу модели.

## Примеры

```python
from tests.tools.profiler.models.reports import extensions
```
