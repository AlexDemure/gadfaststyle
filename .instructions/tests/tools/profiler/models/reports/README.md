## Описание

`tools/profiler/models/reports` описывает модели отчетов профилировщика.

## Правила

- Здесь держи только структуры данных отчетов.
- Форматы отчетов по языкам и расширениям раскладывай по подпапкам.
- Не смешивай модели отчетов с генерацией самих отчетов.

## Примеры

```python
from tests.tools.profiler.models import reports
```
