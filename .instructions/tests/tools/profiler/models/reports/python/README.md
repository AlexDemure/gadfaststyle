## Описание

`tools/profiler/models/reports/python` описывает Python-модели отчетов профилировщика.

## Правила

- Здесь держи только Python-specific структуры отчетов.
- Не смешивай их с общими моделями других языков и форматов.
- Модели должны оставаться техническими.

## Примеры

```python
from tests.tools.profiler.models.reports import python
```
