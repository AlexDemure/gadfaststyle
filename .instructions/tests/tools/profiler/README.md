## Описание

`tools/profiler` описывает профилировщики и отчеты тестового слоя.

## Правила

- В `profiler` держи общие инструменты измерения и отчетности.
- Контекстные менеджеры, модели, профилировщики и отчеты раскладывай по подпакетам.
- Профилирование не должно менять поведение тестируемого сценария.

## Примеры

```python
from tests.tools.profiler import reports
```
