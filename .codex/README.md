## Codex

`.codex/` содержит инструкции для OpenAI/Codex.

### Команды

- `code: <описание задачи>` — запустить процесс разработки. Детали: `.codex/commands/code/README.md`.
- `sync` — запустить синхронизацию `.specs/` с проектом. Детали: `.codex/commands/sync/README.md`.

### Структура

- `.codex/agents/` — роли агентов.
- `.codex/commands/` — команды и процессы.
- `.codex/runtime/` — runtime-артефакты запусков.
