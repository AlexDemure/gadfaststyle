## Назначение

`.codex/agents` хранит агентов Codex и описание их ролей.

## Состав

- [system-analyst/README.md](./system-analyst/README.md) - ведет `spec.md` в `.specs/`.
- [backend/README.md](./backend/README.md) - вносит изменения в `src/` по `spec.md`.
- [tester/README.md](./tester/README.md) - вносит изменения в `tests/` по `spec.md`.
- [team-lead/README.md](./team-lead/README.md) - проверяет результат и делает code review.
- [spec-sync/README.md](./spec-sync/README.md) - синхронизирует `.specs/` с `src/` и `tests/`.
