## Назначение

`.codex/commands` хранит команды Codex и описание логики их выполнения.

## Состав

- [code/README.md](./code/README.md) - полный цикл разработки: от бизнес-задачи до spec-файлов, генерации кода, тестов и финального отчета.
- [sync/README.md](./sync/README.md) - синхронизация `.specs/` с текущим состоянием `src/` и `tests/`.
